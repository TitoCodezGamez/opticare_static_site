# Opticare static site build

This folder is a fully static export of the Opticare site.

## Pages
- `/index.html`
- `/locations/index.html`
- `/locations/<loc-id>/index.html`

## Assets
- `/static/core/css/site.css`
- `/static/core/js/site.js`
- `/static/img/...`

## Local preview
Any static server works:
- `python -m http.server 8000` and visit http://localhost:8000
